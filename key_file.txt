## Key Directories and Their Functions

- **.github/workflows/**: Contains CI/CD and deployment configurations.
- **data/**: Stores raw data files for analysis.
- **external_data/**: Houses external datasets, including geographical data.
- **project/**: Contains Python backend code for data analysis.
- **public/**: Holds static assets and the HTML template for the React app.
- **results/**: Stores output data from various analyses.
- **scripts/**: Contains utility scripts like data preparation.
- **src/**: Houses all React components, hooks, utilities, and styles.

## Key Files and Their Functions

- **config-overrides.js**: Provides configuration overrides for Create React App.
- **package.json**: Defines project dependencies and scripts.
- **requirements.txt**: Lists Python dependencies for backend scripts.
- **src/App.js**: Main React component, manages overall app state and layout.
- **src/Dashboard.js**: Primary dashboard component with responsive layout.
- **src/index.js**: Entry point for the React application.

## Utility Directories

- **src/components/**: Contains reusable React components organized by feature.
- **src/hooks/**: Houses custom React hooks for data fetching and state management.
- **src/styles/**: Contains styled-components and theme configurations.
- **src/utils/**: Includes utility functions and configurations for the frontend.

## Analysis Scripts

- **project/ecm_analysis/**: Scripts for Error Correction Model analysis.
- **project/price_differential_analysis/**: Scripts for analyzing price differentials.
- **project/spatial_analysis/**: Scripts for spatial analysis and visualization.
